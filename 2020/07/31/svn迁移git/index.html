#git仓库迁移

将svn转移到git并保留提交记录

  • 1.确定svn所有用户

    svn log --xml | grep author | sort -u | perl -pe 's/.*>(.*?)<.*/$1 = /' >user.txt

    示例：svn=git<email>
    wangdekun = wangdekun<dekun555@163.co>

  • 2.拉取svn代码，并转换成git版本控制
git svn clone  --no-metadata --no-minimize-url --authors-file=users.txt  [svn url]  [local file name]

示例
 
 git svn clone  --no-metadata --no-minimize-url --authors-file=users.txt  https://127.0.0.01/svn/repo/code  iotcode
  • 3.清理svn信息参考官方文档 git教程 。注：也许不需要清理
  • 4.本地仓库添加添加远程仓库
 添加
git remote add [origin name] [url]

 查看
 git remote -v

  • 5.拉取远程仓库，因为本地仓库与远程仓库无交集，需要使用git pull origin master --allow-unrelated-histories 拉取代码。参考

  • 6.推送到远程服务器git push origin --all
