Mysql基础-3 ---『 运算符 』

本文最后更新于：2021年7月22日 下午

正文的一部分作为摘要

Mysql运算符

算数

运算符 描述
+ 加法
- 减法
* 乘法
/ 或 DIV 除法 (div 只返回商)
% 或 MOD 取余

除法或取余 运算 除数为0，返回结果： null

比较

  • 数字比较，浮点数比较
  • 字符串比较，不区分大小写比较
  • 表达式比较
  • true为1，false为0
运算符号 描述
= 等于
<> 或 != 不等于
📌 <=> null 安全的等于（null-safe） 【可与参与null的比较】
< 小于
<= 小于等于
> 大于
>= 大于等于
between 存在于指定 范围 ; 格式：field between min or max,闭区间
in 存在与指定 集合 ; 格式：field in(value1,...,valueN)
is null 是null
is not null 不是null
like 通配符匹配【百分号（%）、下划线（_）】
regexp 或 rlike 正则匹配

逻辑

运算符 描述
not 或 ! 非 ； 备注：not null ⇒null
and 或 && 与 ； 备注：任意一个操作数是null 则 ⇒ null
or 或 || 或 ； 备注： 两个均为null 则 ⇒ null；有一个null，另一个非零则 为true，否则 null
xor 异或 ；相同为0，不同为1； 备注：任意一个操作数是null 则 ⇒ null

位

运算法 描述
& 位与（位AND）
｜ 位或（位OR）
^ 位异或（位XOR）
～ 位取反
>> 位右移 「左补0」
<< 位左移「右补0」

优先级

优先级顺序「优先级从低到高 ↓ 」 运算符
1「优先级最低」 :=
2 ||、OR、XOR
3 &&、AND
4 NOT
5 BETWEEN、CASE、WHEN、THEN、ELSE
6 =(比较运算)、<=>、<=、>、>=、!=、<>、IN、IS NULL、LIKE、REGEXP
7 ｜
8 &
9 <<和>>
10 -和+
11 *、/、DIV、%和MOD
12 ^
13 -(负号)和 ～
14 「优先级最高」 ！

优先级随便看看就好，使用 （） 控制优先级。

= 与 :=（赋值） 区别[1]

  • =
    • 只有在set和update时才是和:=一样，赋值的作用，其它都是等于的作用。鉴于此，用变量实现行号时，必须用:=
  • :=
    • 不只在set和update时时赋值的作用，在select也是赋值的作用。
